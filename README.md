# PersianDateJs

A gem for adding PersianDate(v0.1.8) JS library to your Rails project.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'persian_date_js'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install persian_date_js
